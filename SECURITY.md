# Security Policy

## Supported versions

This project is currently maintained on the latest default branch.

## Reporting a vulnerability

If you discover a security issue:

1. Do not open a public issue with exploit details.
2. Contact the maintainer privately first.
3. Provide clear reproduction steps and impact.

## Scope

Potential security concerns include:

- accidental credential leaks in commits
- unsafe handling of untrusted URLs
- risky file output behavior
